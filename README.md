# Smart Device System (Java OOP Project)

This is a simple Java project that demonstrates core **Object-Oriented Programming (OOP)** concepts using a smart device system.

---

## 📌 Concepts Covered

- Inheritance (`extends` keyword)
- Superclass vs Subclass
- Single Inheritance
- Multilevel Inheritance
- Hierarchical Inheritance
- Method Overriding
- `this` keyword
- `super` keyword (constructor & methods)
- `super()` constructor calls
- Protected access modifier
- Static methods & Method Hiding
- Final method
- Final class
- Multiple Inheritance Restriction in Java

---

## 📁 Project Structure
SmartDeviceSystem/
├── SmartDevice.java
├── Devices.java
└── Main.java

---

## ▶️ How to Run

1. Open terminal in the project folder  
2. Compile:


javac *.java


3. Run:


java Main

## 🧠 Notes

- Java does **not support multiple inheritance** with classes  
- `final` class cannot be extended  
- `final` method cannot be overridden  
- Static methods demonstrate **method hiding**, not overriding
